# PLAN: Skill Self-Crystallization

Priority: P0 - BLOCKING
Created: 2026-04-25
Status: READY FOR IMPLEMENTATION

## Goal
Implement autonomous skill crystallization that captures successful task execution paths and makes them reusable.

## Core Pattern (GenericAgent)
[Task Complete] --> [Analyze] --> [Generate SOP] --> [Store] --> [Recall]

## Architecture
~/.meowju/memory/L3-skills/skill-{uuid}.md

## Implementation Phases

### Phase 0: Prerequisites
- EPOCH 14: Auto-approve learning (approvalCount, dangerous patterns)
- EPOCH 17: WAITING state in AgentState

### Phase 1: Skill Store
- src/skills/skill-store.ts
- CRUD for skills
- FTS5 search for findSimilar()

### Phase 2: Execution Tracker
- src/skills/execution-tracker.ts
- Track tool calls during session
- startSession(), recordToolCall(), endSession()

### Phase 3: Crystallizer
- src/skills/skill-crystallizer.ts
- shouldCrystallize() - checks 3+ tools, success, not duplicate
- crystallize() - LLM extracts name/trigger, saves skill

### Phase 4: Recaller
- src/skills/skill-recaller.ts
- beforeTask() - finds similar skills, returns match + suggestion

### Phase 5: Agent Integration
- Hooks in lean-agent.ts for:
  - Tool call tracking
  - Post-completion crystallization
  - Pre-task recall

## Test Criteria

### Unit Tests
- SkillStore.save() / findSimilar()
- ExecutionTracker lifecycle
- Crystallizer.shouldCrystallize() / crystallize()
- Recaller.beforeTask()

### Integration Tests
- Full flow: session -> skill -> store -> retrieve
- Recall on similar prompt
- Usage count increments

### Dogfood Validation
1. meow "Create react app called test-app"
2. Complete task (3+ tools)
3. Verify skill file exists
4. meow "Make react project named demo"
5. Verify "I remember" message

## Acceptance Criteria
- [ ] Skills stored as markdown in L3-skills/
- [ ] Similar prompts recall skills (confidence > 0.7)
- [ ] Skill execution increments usageCount
- [ ] Only 3+ tools + success = crystallized
- [ ] Skills persist across restarts
- [ ] Integration tests pass
